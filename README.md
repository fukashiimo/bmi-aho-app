# BMI毒舌判定機

身長と体重を入力すると、BMIを細かめに分岐して「痩せすぎー」「標準じゃん！めっちゃいいじゃん！」「太り過ぎー」などを表示する静的Webアプリです。

## GitHub Pagesで公開する

1. このリポジトリをGitHubにpushします。
2. GitHubのリポジトリ画面で `Settings` → `Pages` を開きます。
3. `Build and deployment` の `Source` を `Deploy from a branch` にします。
4. `Branch` を `main`、フォルダを `/ (root)` にして保存します。

公開URLは通常 `https://<ユーザー名>.github.io/<リポジトリ名>/` になります。
